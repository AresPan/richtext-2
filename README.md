# richtext

富文本（也就是我们常常说到的图文混排）经常在我们做app应用的时候要求做到这个功能。 
比如说，

做一个论坛的app时候我们需要发帖，而发帖就要求我们做到图片和文字一起展示，发送到服务器中去，
做某些记事本的app的时候，我们也要对做的富文本进行操作，把富文本存储到本地，以便于我们查看和重现编辑
我做了一个app，先把平常操作的图贴上，主要功能是编辑富文本，和富文本的本地存储，及富文本的二次编辑。
